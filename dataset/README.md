# Dataset

This repository includes six supporting CSV files.

The original `sales.csv` file is approximately **493 MB**, which exceeds GitHub's file size limit and is therefore not included.

To run this project locally, add the original `sales.csv` file to this folder.

Expected folder structure:

```
dataset/
├── sales.csv
├── customers.csv
├── products.csv
├── categories.csv
├── employees.csv
├── cities.csv
└── countries.csv
```
